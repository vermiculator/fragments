* teach people how to use the tech and empower them to do so
* good [[UX]] and [[a11y]]
* and/or use the interface to question, critique, reimagine or invite play with the whole ecosystem
* open source it, copyleft it, dedicate it to the commons
* allow a self-hosted option
* un-silo, decentralise, distribute
* encourage data sovereignty
* build for modularity, decoupling and interoperability
* comment code well, document systems well
* don't [[reinventing the wheel|reinvent the wheel]] but do [[reinvent proprietary wheels]]
